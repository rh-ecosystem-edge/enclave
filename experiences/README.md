# Enclave Experiences

An Experience is a collection of plugins and functionality that together form a customer experience.

| Experience | Description |
|------------|-------------|
| **VMaaS** | VMs as a Service — virtual machine provisioning on OpenShift Virtualization |
| **CaaS** | Containers as a Service — OpenShift cluster provisioning with Hosted Control Planes |
| **BMaaS** | Bare Metal as a Service — bare metal server provisioning and lifecycle management |
